# Minawiki

最初的设想是提供一个匿名为课程/老师评论和评分的地方，为后来同学选课提供参考。后来觉得以维基的方式提供比较好，既有各个课程的介绍，下面又有评论的留言区。寻找了很多引擎没有发现有完善的评论和点赞机制的，就自己开发好了。

Minawiki 就是应用于这样的应用环境的 wiki 引擎，用于 NEU Wiki， *东北大学选课维基*的引擎。

这个引擎大概也可用于别的学校，或者其他应用场景，比如本科专业介绍等等。
